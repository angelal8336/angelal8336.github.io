Movie Webpage Project

Context

We were asked to create a fictonal movie of our choose using basic html and css

Content

For this project me and my partner Jonathan decided to make a movie on the topic octopus. I have no idea why we chose this topic, but we knew we wanted to create an adventure.

So, we came up with, Octopus On The Run. The octopus decided to escape from his "home" one fatal night. The chase for this octopus begin from there on. Will he escape or be capture once again?

Doing this movie alone, would of taken lots of time. However, each of us did part of the work, discussed, and helped one another, which made this project go alot quicker.

Challenges

This project was much harder than I have anticipated. Formatting our website was by far the toughest of them all. We wanted certain images to go certain places but no matter what we tried it always looked off

We of course asked for help and used google, but it wasn't what we had envisioned. With the current knowledge we had, formatting the website like we wanted was out of the blue.

However, in the end we found a format we made that we liked alot.

Takeaways

  1. Working together and communcating was the key to this project all along, as without it, the work could of not been done.
  2. We are still newbies at coding, but we will improve as time goes by
  3. This was fun
