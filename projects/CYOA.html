Choose Your Own Adventure Project

Context

My partner Henry, and I were asked to create our own story based on a topic that was given to us

Content

The topic we were given asked us to create a adventure where you die if you get a answer wrong, and so we did.

This project is formatted as a trivia. It revolves around you being in a school, and answering questions to survive.

Purpose

The purpose of this project was to get us to practice linking, as well as our collabration skills

In the future we will need to know how to collabrate with different people who are working in the same project as us

This project will help us rememeber how to link, and improve our collabration skills

Challenges

The hardest part of this project was communicating with your partner even though it was a team of 2 (including me)

We had to tell each other when to push or pull, and what each of us needed to do

However, that was not difficult compare to resolving merge conflicts

Whenever, we got a merge conflict we had to discuss which one we should keep

The only reason we were able to solve this problem is because we talked it out

Luckily for us, my partner and me were in the same class so it was easier to do so

Next time

If we had more time, we would add more files/pictures so that the adventure can be more cooler

Takeaways

  1. Without communication, it is basically impossible to complete or do well in your project
  2. When you start linking more and more files together, it will become hard to keep track of it
  3. This project hurts my brain
