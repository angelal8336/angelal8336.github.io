Own character design

Context

I been learning how to draw for about one year and a half now. For this project of my own, I created a drawing of a character I came up with using clip studio art. My inital goal was not to make this into a real digital art as I drew my characters within my notebook. It was more of a hobby since I was interested in learning how exactly anime creators drew. It started as a simple doodle until I really liked a design I came up with, and was thinking of trying digital art for a while now, so I gave it a go. In fact this is my first time drawing online, and learning how to use the application was a difficult by itself.

Process

When I decided to transport my notebook drawing onto the computer, I had to learn how clip studio art worked. It wasn't easy per say but I watched tutorial videos, while testing out all the speical features avaiavble for use.
